# IPL-2022-Auction-Analysis

## Overview
Analysis of IPL 2022 auction data using Python and Pandas.

## Key Findings
- 633 players entered the auction, 396 went unsold
- KL Rahul was the most expensive player at 17 CR
- Wicketkeepers commanded the highest average price
- Mumbai Indians spent the most money overall
- CSK bought the most players

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
IPL 2022 Auction Dataset from Kaggle
